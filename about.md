## Featured Project

### domharvest-playwright

[![npm version](https://badge.fury.io/js/domharvest-playwright.svg)](https://www.npmjs.com/package/domharvest-playwright)
[![GitHub](https://img.shields.io/github/stars/domharvest/domharvest-playwright?style=social)](https://github.com/domharvest/domharvest-playwright)

A powerful DOM harvesting tool built with Playwright for extracting and analyzing web content. It simplifies the process of collecting structured data from websites with features like:

- Smart CSS selector discovery
- Automatic content extraction
- Screenshot and PDF generation
- Headless and headed mode support
- Extensive configuration options
